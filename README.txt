You need a special dependency to run the game. 
    (1) Python 


Instructions
==============
1. Install Python, via any means (you can download from any internet browser and install) 
2. Open the "Heritage Game" folder. 
3. Open a terminal or command prompt inside the folder. 
4. Type "python -m http.server 8080". 
5. Open browser, goto "http://localhost:8080" , the game will run here. 



In short, 
The game just needs a server to run on, give it a quick one, a python server.
